üîß Meet Nmap

Nmap: The "Network Mapper" - the most famous network scanning tool in the world! Our first step is to find which machines are active on the network. We can do this with a simple "ping scan".

This command sends a ping to every possible address in the range and lists the ones that respond.

üéØ Understanding Password Attacks

Brute-force attacks involve repeatedly guessing passwords until the correct one is found. Common strategies include dictionary attacks (using known words or phrases) and exhaustive attempts with every possible character combination.

üß∞ Meet Hydra & Crunch

Hydra is a fast network login cracker. It supports a wide range of protocols like FTP, SSH, HTTP, RDP, and more. Hydra is ideal for online brute-force attempts against exposed login services.


Crunch is a wordlist generator that creates custom dictionaries by combining and permutating characters. You can define the exact charset, length, and pattern of the words it produces, making it perfect for generating targeted lists based on known password structures.


üîé Bonus Tools (Not covered in this workshop)

John the Ripper is a powerful offline password cracker designed to target hashed credentials. It supports a wide variety of hash types and uses smart wordlist mangling and rule-based attacks.


Hashcat is a GPU-accelerated password recovery tool that's extremely efficient for cracking complex hashes at scale. It's commonly used for offline brute force and dictionary attacks on large datasets.

üí° From Brute Force to Strategy

To understand how quickly brute-force attacks scale, let's generate some wordlists using Crunch. You‚Äôll create lists of all lowercase character permutations for 3, 4, and 5-character passwords.

Start with 3 characters:

crunch 3 3 abcdefghijklmnopqrstuvwxyz -o 3letter.txt

Then generate 4-character permutations:

crunch 4 4 abcdefghijklmnopqrstuvwxyz -o 4letter.txt

And finally, 5-character permutations:

crunch 5 5 abcdefghijklmnopqrstuvwxyz -o 5letter.txt

Once generated, compare the file sizes and number of lines (passwords) for each: 

ls -lh 3letter.txt 4letter.txt 5letter.txt
            wc -l 3letter.txt 4letter.txt 5letter.txt


With 26 lowercase letters, here's what's happening under the hood:

  • 3-letter: 26¬≥ = 17,576 permutations
  • 4-letter: 26‚Å¥ = 456,976 permutations
  • 5-letter: 26‚Åµ = 11,881,376 permutations

That's exponential growth for only lowercase letters. Add uppercase, numbers, or symbols and the space expands even faster.


This is why exhaustive brute-force lists are rarely practical. Instead, a smarter strategy is to start with common or default passwords. Many accounts still use simple values like admin, password, or welcome123. Trying these first is often faster and far more effective than crunching millions of possibilities.

üîß Creating Smart Permutations

Instead of blindly generating every possible permutation, you can create smarter wordlists by modifying existing words with common patterns, prefixes, and suffixes. Tools like rsmangler automate this process.

  • Generate a smart wordlist using rsmangler:

    rsmangler creates permutations by intelligently modifying words from an existing wordlist. For example, to prepend or append specific values to a base wordlist: 

    rsmangler --file wordlist.txt --prefixes foo,bar,foobar --suffixes xor --output mangled_wordlist.txt

    This command achieves:

    • Prepending foo, bar, and foobar to each word
    • Appending xor to each word
    • Saving the result to a file named mangled_wordlist.txt
  • Examine the mangled wordlist:

    Verify the contents and size of the generated wordlist:

    cat mangled_wordlist.txt

üì° Setting Up Monitor Mode

Monitor mode explained: Normal WiFi cards only see traffic meant for them. Monitor mode lets you see ALL WiFi traffic in the area - like turning your WiFi card into a "WiFi wiretap".


Check available WiFi interfaces
airmon-ng

This shows your WiFi adapters and their current status

Enable monitor mode
airmon-ng start wlan1

This puts your WiFi adapter into monitor mode (interface becomes wlan1mon)

Scan for WiFi networks
airodump-ng wlan1mon

This shows all WiFi networks in range with details like encryption type and signal strength


üîç What to look for: Target networks with WPA/WPA2 encryption and good signal strength. Avoid networks with WPS enabled as they have other vulnerabilities.

üéØ Capturing the WPA Handshake

The handshake: When a device connects to WiFi, it performs a "4-way handshake" with the router. This contains encrypted password data we can crack offline.


Target specific network
airodump-ng -c [CHANNEL] --bssid [ROUTER_MAC] -w capture wlan1mon

Focus on one network and save captured data to files

Force a handshake (deauth attack)
aireplay-ng -0 5 -a [ROUTER_MAC] -c [CLIENT_MAC] wlan1mon

Disconnect a client to force re-authentication (and capture the handshake)

Verify handshake capture

Look for "WPA handshake" message in airodump-ng output


‚ö° Pro tip: You need at least one device connected to the target network to capture a handshake. The deauth attack temporarily disconnects them, forcing a new handshake.
